# Coder of Bangalore

## Project Overview

This project parses raw Instagram profile data from a text file and performs analysis to extract insights such as:

- User with the maximum number of posts
- User with the maximum number of followers
- User who follows the most accounts
- Count of account categories

The project demonstrates practical data-cleaning and analysis techniques using Python.

---

## Files Included

- `project_code.ipynb` – Main analysis notebook
- `project_description.ipynb` – Project scenario and background
- `instagram_varied_data.txt` – Raw dataset
- `README.md`

---

## Features

- Parse semi-structured text data
- Convert raw text into Python dictionaries
- Transform data into JSON
- Analyze Instagram metrics
- Count categories and summarize results
